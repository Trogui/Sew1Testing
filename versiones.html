Breve historia

React fue creado por Jordan Walke, un ingeniero de software en Facebook, que lanzó un prototipo inicial de React llamado FaxJS; fuertemente influenciado por XHP, una biblioteca de componentes HTML para PHP, la primera vez que se utilizó fue en las noticias de Facebook en el 2011, después en Instragram en el 2012 y liberado (código abierto) en el 2013. En el 2015 Facebook presenta React Native, nos va a permitir el desarrollo nativo de Android, iOS y UWP. En el 2017 Facebook presenta React Fiber, es un algoritmo para consumir interfaces de usuario, que más tarde se convertiría en la base de React. De las últimas actualizaciones, quizás una de las más importantes es en el 2019, donde introduce Hooks.
